VisualVM 1.0 Release Notes

VisualVM is a visual tool integrating several commandline JDK tools and lightweight profiling capabilities. Designed for both production and development time use, it further enhances the capability of monitoring and performance analysis for the Java SE platform.

System Requirements

Supported OS:

  • Microsoft Windows (XP, Vista, Server): 32bit & 64bit
  • Linux: Intel platform, 32bit & 64bit
  • Sun Solaris: Intel & Sparc platform, 32bit & 64bit
  • Mac OS X: Intel platform, 64bit

Required software: Sun or Apple JDK 6+

Known Problems

  • Issue 152: VisualVM encountering problems with JDK 7 b27 and newer
  • Issue 120: JMX application datasource cannot be reused on application restart
  • Issue 72: Running application sometimes not recognized
  • Issue 91, Issue 37: Close buttons missing or incorrectly displayed in subtabs on GTK LaF
  • Issue 53: No action on Proxy Configuration button

To view all known problems please use this Issue Tracker query.
