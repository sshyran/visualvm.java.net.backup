Prerequisites

Be sure that JDK 6.0 or 7.0 is correctly installed on your system. Whereas VisualVM is able to monitor JDK 1.4+ applications, it requires JDK 6.0+ as host Java platform. Note that VisualVM has been only tested to run on Sun's JDK. See SYSTEM REQUIREMENTS for more details.
