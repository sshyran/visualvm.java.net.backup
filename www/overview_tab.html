Application Overview

VisualVM presents data for local and remote applications in a tab specific for that application. When you open an application in VisualVM, each application opens in a new tab in the main window. You can have multiple application tabs open.

Each application tab contains sub-tabs that display different types of information about the application.

The Overview tab displays general information about the application and the runtime environment. Application data is displayed in the following sub-tabs.

The Overview sub-tab displays the following general information specific to the application:

  • PID.
  • Host
  • Main class
  • Arguments. Displays any options passed to the application on startup.
  • JVM
  • Java Home
  • JVM flags
  • Heap dump on OOME. This displays the status of the Heap Dump on OOME option. You can enabled or disable the option in the application popup menu.

The following sub-tabs provide information:

  • Overview. This sub-tab displays general information specific to the application.
  • Saved Data. This sub-tab displays application information that VisualVM has stored about the application.
  • JVM Arguments. This sub-tab displays information about arguments passed to the JVM.
  • System Properties This sub-tab displays information about the system that the JVM is running on.
